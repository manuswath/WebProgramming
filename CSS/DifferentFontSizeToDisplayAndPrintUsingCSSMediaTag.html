The @media Rule

The @media rule makes it possible to define different style rules for different media types in the same stylesheet.

The style in this example tells the browser to display a 17 pixels Verdana font on the screen. But if the page is printed, it will be in a blue 14 pixels Georgia font.

Test it! Print this page (or open Print Preview), and you will see that the text will be displayed in blue, and in another smaller font.
